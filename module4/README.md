# Module 4 – Dashboard Planning and Prototyping

## Project Overview

This project is developed as part of the **Infosys Internship – Module 4: Dashboard Planning and Prototyping**.

The application is a **Streamlit-based Military Power Dashboard Prototype** that allows users to explore military data through interactive visualizations, KPIs, and filters. The prototype demonstrates dashboard planning, navigation, and data visualization using the cleaned military dataset.

---

## Dashboard Features

### Quick Stats
- Displays key military KPIs
- Military Power Index
- Global Rank
- Defense Budget
- Active Personnel
- Military Assets

### Nation Overview
- Country-wise military information
- Interactive charts
- Summary statistics

### Compare Powers
- Compare military strength between countries
- Visual comparison using charts

### Coalition Builder
- Prototype page for analyzing combined military capabilities of selected countries
