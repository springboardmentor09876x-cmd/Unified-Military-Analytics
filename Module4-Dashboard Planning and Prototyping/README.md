# 🛡️ Unified Military Analytics Dashboard

A comprehensive **Streamlit-based Military Analytics Dashboard** that enables interactive analysis and comparison of military capabilities across countries using data visualization and filtering.

## 🌐 Live Demo

**Deployment Link:**  
https://military-dashboard-deploy-hi3mgs6z88ivcn7cqpdfbs.streamlit.app

---

## 📌 Features

- 🌍 Interactive country, continent, and region filters
- 📊 Military capability comparison dashboard
- 📈 Multiple interactive Plotly visualizations
- 💰 Defense budget and GDP analysis
- ✈️ Aircraft, 🛡️ Tanks, 🚢 Naval Fleet analysis
- 🌎 World map visualization
- 📋 Searchable military dataset
- 📥 Download filtered dataset as CSV
- 🎨 Responsive dark-themed UI using custom CSS

---

## 🛠️ Tech Stack

- **Python**

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/Nagashreemurthy21/military-dashboard-deploy.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
