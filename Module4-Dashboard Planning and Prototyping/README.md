# 🛡️ Unified Military Analytics Dashboard

## 🌍 Global Defense Intelligence Dashboard

An interactive Streamlit dashboard that provides comprehensive analysis and comparison of global military capabilities using defense-related datasets.

The dashboard enables users to explore military strength, defense resources, air power, land forces, naval assets, and strategic capabilities across different countries.

## 📌 Project Overview

The Unified Military Analytics Dashboard transforms raw military data into meaningful visual insights through interactive filters and advanced visualizations.

Users can analyze:

- Country-wise military capabilities
- Continental and regional comparisons
- Defense strength indicators
- Military personnel distribution
- Air, land, and naval power statistics
- Strategic capability comparisons

---
